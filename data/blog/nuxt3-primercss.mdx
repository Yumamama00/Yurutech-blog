# Primer CSS とは

[Primer CSS](https://primer.style/css/)とは、GitHub のデザインシステムを支える CSS フレームワークです。<br />
Primer CSS を利用することで、GitHub と同様なデザインのコンポーネントを簡単に作成することが出来ます。<br/>
以下は、実際に Primer CSS のボタンコンポーネントを利用したボタンです。GitHub のボタンそのまんまですね。

ボタン以外にも、アラートやフォーム、ナビゲーション、パンくずリスト、ダイアログ等、Web アプリケーションで実装する機会の多いコンポーネントが豊富に用意されています。

※ Primer CSS では、JavaScript は提供されていません。一部の処理は独自で実装する必要があります。<br />
※ ただし、[React のコンポーネント](https://primer.style/react/)は提供されているようです（Vue 版も欲しい、、）。

# Nuxt3 で試す

前記の通り、React のコンポーネントは提供されていますが、Vue.js の実装は提供されていない為、CSS のみを利用する形になります。

導入自体は至ってシンプルです。

1. Primer CSS をインストール

```shell
npm install @primer/css --save
yarn add @primer/css
```

2. assets ディレクトリに `primer.scss` を作成

```scss
@import '@primer/css/index.scss';
```

3. `nuxt.config.js/ts` に以下を追記

```conf
css: ['@/assets/primer.scss']
```

4. 実際に CSS が適用されているか試す

```html
<button class="btn btn-primary mr-2" type="button">Primary</button>
```
